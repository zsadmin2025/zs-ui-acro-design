
# zs-project-java

## 项目介绍
ZS-Admin是一套基于前后端分离的快速开发脚手架。基于JDK17、Spring Boot 3.X、Spring Security、MyBatis-Plus、JWT、Vue3、Acro Design vue、vite、pinia构建。
